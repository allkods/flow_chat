## Prerequisites
### STEP 1
- Edit line 41 of `app.js`
- Pass your own mongoDB connection string to `mongoose.connect()` parameter and save it


## Running the server

To Install all the dependencies
```bash
    npm i
```

To start server
```bash
    node app.js
```
